Misskeyをはじめよう

いつまでも進化する、洗練されたマイクロブログSNS Misskey（ミスキー）

Misskeyマスコット 藍（Ai）

Misskeyを見つけていただき、ありがとうございます。Misskeyは、地球で生まれた分散マイクロブログSNSです。Fediverse（様々なSNSで構成される宇宙）の中に存在するため、他のSNSと相互に繋がっています。

暫し都会の喧騒から離れて、新しいインターネットにダイブしてみませんか。

はじめるよくある質問運営する

Misskeyをはじめる理由

普段使いのSNSがMisskeyでなければならない理由は、たくさんあります。

多くの機能

Misskeyでできることはたくさんあります。

Wikiで使い方を見る

充実した投稿機能

インスタンスごとに設定された自由な文字数制限

公開範囲を投稿ごとに細かく設定

長い文章や不適切な画像を隠して公開

ノートの詳細

文字装飾

投稿やプロフィールを装飾しよう。装飾構文(Misskey Flavored Markdown)を利用できます。

MFM構文一覧

リアクション

投稿への共感を簡単に表現しよう。絵文字を選んで、投稿にリアクションを送れます。

デッキ

自由に画面を作ろう。Misskeyのデッキは、タイムラインを縦に並べることもできます。

アンケート

みんなに意見を聞いてみよう。10個までの選択肢でアンケートを作成できます。複数回答可や有効期限付きにできます。

ドライブ

ファイルを保存して何回も共有しよう。添付したファイルをドライブで管理したり、ドライブにあらかじめファイルをアップロードしたりできます。

ドライブの詳細

チャット

チャットしよう。
ノートとは別の専用画面で、一対一もしくはグループでチャットができます。

チャットの詳細

ページ

ジェネレーターを作ったり、文章を書いたりしよう。
ページでは、テキストやセクションといったブロックを配置し、簡単なプログラムを組めます。

ページの詳細

ルーム

いろいろな家具を配置して自分の部屋を作ろう。
作成したルームはURLでみんなと共有できます。

ルームの詳細

アンテナ

エゴサしよう。
アンテナ機能で設定した条件を含むノートを収集できます。

チャンネル

集まる場所ができました。
様々な話題について、チャンネルを作って会話することができます。

カスタマイズ性の高い画面

MisskeyのWebクライアントはあなたの好みで自在に設定できます。

ウィジェット

小さなウィジェットは様々な情報を表示します。ウィジェットを自由に選んで並べ替えることができます。

テーマ

Misskey Webの背景や文字、枠線などの色を自由に変更できます。
作成したテーマはファイルとして保存・共有できます。

Assets Store

独立性と社会性

あなたの意志で運営者を選択する、もしくはあなた自身でサーバーを運営することもできます。

Fediverse

MisskeyはActivityPubというWeb標準規格のサーバー間通信機能を持っているため、 違うインスタンスや他のサービスのユーザーをフォローしたり投稿を見て反応したりできます。

簡単なインスタンス構築

Misskeyのインストールは簡単です。自分のSNSを持つことができます。

インスタンスの構築・管理

オープン

Misskeyはオープンソースソフトウェアです。誰でも開発や翻訳に参加できます。リポジトリをフォークし、独自の拡張を追加することもできます。

GitHubCrowdin(翻訳)

API

アプリやクライアントを作って生活に役立てよう。MisskeyのほぼすべてのAPIエンドポイントを外部から利用できます。リバーシBotと対戦し、今聞いている曲をみんなに知らせましょう！
